Import Guide

Upload an Excel (.xlsx, .xls) or CSV (.csv) file with insurance payment data. The system will automatically match columns based on the following names:

Patient/Member Name:
Member Name, Patient Name, Patient
Amount (Priority):
Claim amount paid, Check/EFT amount, Amount
Member ID:
Member Subscriber ID, Member ID, Subscriber ID
Check Number:
Check/EFT number, Check Number, Check #, EFT Number
Payment Date:
Payment date, Payment Date
Claim Status:
Claim status, Claim Status
Amount:
Check/EFT amount, Amount, Payment Amount
Payee Name:
Payee name, Patient Name, Name
Claim Number:
Claim number, Claim Number, Claim #
Payee Address:
Payee address, Address

Tip: Column matching is case-insensitive.
